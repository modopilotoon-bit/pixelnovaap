# PixelNova OS

Sistema operativo de trabajo para Pixel Nova — estudio de marketing digital en Cancún.

## Instalación local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

## Deploy a Netlify

1. Push a GitHub
2. Conectar repo en netlify.com
3. Build command: `npm run build`
4. Publish directory: `dist`

## Instalar como PWA

- **iPhone**: Safari → Compartir → Añadir a pantalla de inicio
- **Android**: Chrome → Menú → Instalar app / Agregar a pantalla de inicio
